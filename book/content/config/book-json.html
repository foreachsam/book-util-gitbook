GitBook Entrance

book.json

「book.json」

編輯「book.json」

$ vi book.json

貼入下面的內容

{
    "gitbook": ">=2.0.0",
    "title": "demo book",
    "description": "learn gitbook"
}

以上是基本的設定範例，

執行下面指令，

$ gitbook serve

觀看書本網頁內容，觀看「Introduction」那一頁，你會發現，最上面的表題會是「Introduction | demo book」。

或是觀看網頁原始碼，你會看到

<title>Introduction | demo book</title>

關於「description」，觀看網頁原始碼則會看到

<meta name="description" content="learn gitbook">
