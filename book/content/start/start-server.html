觀看網頁書本內容

執行

$ firefox http://localhost:4000

或是

$ lynx http://localhost:4000

改用不同的Port啟動Server

啟用「Server」,預設使用的「Port」是「4000」。

下啟用「Server」指令的時候，可以使用「--port 4001」，指定不同的「Port」來啟動。

$ gitbook serve --port 4001
